# my_project

## Project setup

### 克隆项目

```
git clone https://github.com/zsj975516/electron-vue-cli3.git
```
 
 ### 安装依赖
```
cd electron-vue-cli3

npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
